# MeDeA 审稿问题对应解决方案

## 使用原则

这份文档不讨论“大规模重构项目”，只针对当前审稿意见，一条问题对应一套解决方案。每条都包含：

- 问题是什么
- 应该怎么解决
- 在你当前项目里具体做什么
- 最终在论文里体现成什么

## 问题 1：类别不平衡处理说明不足

### 问题来源
- Reviewer 1 指出：多标签任务用 BCE，但没有说明类别不平衡如何处理。

### 对应解决方案
- 不需要重构训练框架。
- 只需要补一组“类别不平衡处理说明 + 小规模对照实验”。

### 具体动作
1. 统计 `processed_data/` 十折数据中每个类别的阳性比例。
2. 在主模型上补一个对照：
   - `BCEWithLogitsLoss()`
   - `BCEWithLogitsLoss(pos_weight=...)`
3. 重点比较 HYP、MI、CD 这几个难类的 F1 变化。

### 论文落地
- 方法部分补充 `pos_weight` 的定义公式。
- 实验部分加一句是否采用 class weighting。
- 如果加权没有明显提升，也要写明“尝试过，但最终未采用”。

## 问题 3：多个 learnable query 的来源与含义不清楚

### 问题来源
- Reviewer 2 质疑：正文说 query 是可学习向量，但图看起来像是由输入特征生成的。

### 对应解决方案
- 不改代码逻辑。
- 直接澄清：query 是模型参数，不是样本自适应生成。

### 具体动作
1. 在论文里明确对应到当前实现：
   - `self.queries = nn.Parameter(torch.randn(num_queries, d_model))`
2. 图中删除任何会让人误以为 query 来自输入特征的箭头。
3. 如果论文里有温度参数 `τ`，补上取值；如果代码里没用 `τ`，就从论文中删掉该符号或解释为标准缩放项 `1/sqrt(d_model)`。

### 论文落地
- 方法公式旁写清：
  - query 是全局可学习参数
  - 不依赖单个样本动态生成
- 图 2 必须和代码一致。

## 问题 4：attention 平均与上采样如何形成最终解释不清楚

### 问题来源
- Reviewer 1 认为解释图构建过程不清晰。

### 对应解决方案
- 不改解释方法本身。
- 只补“从 feature-level attention 到 signal-level heatmap”的流程说明。

### 具体动作
1. 明确当前代码流程：
   - `attn_probs` 先在 query 维平均
   - 得到长度为 `L'` 的 attention
   - 用 `np.interp` 上采样到原始信号长度
   - 用 `gaussian_filter1d` 平滑
2. 在论文中用 3-4 句把这个后处理链条写清。
3. 图注中注明“upsampled for visualization only”。

### 论文落地
- 可解释性部分单独写一个 “Attention visualization pipeline” 小节。

## 问题 6：阈值策略不统一，影响比较公正性

### 对应解决方案
- 不改训练方式。
- 只统一测试阶段阈值协议。

### 具体动作
1. 给 MeDeA 主模型补一个与 baseline 相同的 validation threshold search。
2. 主表统一使用一种阈值协议。

### 论文落地

  - “For all multi-label methods, per-class thresholds were tuned on the validation set.”

## 问题 8：MSW-TN 某些指标优于 MeDeA，需要解释

### 问题来源
- Reviewer 2 明确要求分析性能差距。

### 对应解决方案
- 不需要把 MeDeA 强行写成全面最优。
- 需要改论文叙事。

### 具体动作
1. 在结果分析中单独增加一段：
   - 哪些指标上 MSW-TN 更强
   - MeDeA 为什么没有占优
   - MeDeA 的优势在哪里
2. 优势可落在：
   - intrinsic interpretability
   - per-disease attention decomposition
   - 更清晰的病种级解释
3. 如果有效率数据，也可说明复杂度优势或部署优势。

### 论文落地
- 把“最好”改成“competitive”或“offers a stronger interpretability-performance trade-off”。

1. 统一论文主模型与实验协议
