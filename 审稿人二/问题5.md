# 问题 5：稿件没有提供足够证据证明不同查询捕获了不同的形态学模式

## 问题

论文假设不同查询能够捕获不同模式，但还需要更全面的可视化证据，例如展示不同样本上的逐查询注意力图。

## 回复

感谢审稿人指出这一关键问题。我们同意，原稿中关于“不同查询捕获不同形态学模式”的证据仍然不够充分。原因在于，当前正文主要展示的是对查询维度取平均后的类别级注意力图，而不是系统性的逐查询可视化，因此这些主图本身不足以单独证明不同 query 已形成稳定、明确分工的形态学专门化。现有结果更直接支持的是：相较于 single-query 对照，multi-query 设计更有利于形成可分解的解释表示，并带来稳定但有限的性能增益；这一点与当前修订稿中的可视化对比和 single-query 对照结果是一致的。基于这一证据边界，我们已将相关表述收紧为一种定性判断：逐查询可视化现象提示不同 query 可能关注不同的形态学证据，但这仍需更系统的 per-query visual panels 或统计分析进一步验证，而不应被表述为无需限定的绝对结论。
